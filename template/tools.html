Herramientas

Twig es un motor de plantillas para PHP que viene con Symfony2, permite facilmente la reutilización de plantillas mediante tres funciones principales extends, include y embed. Es Rapido, Seguro y Flexible.

Jade es un motor de plantillas para HTML, lo utilizo principalmente para hacer prototipos y diseño de flujo de interfaces para mostrar al cliente.

Less Css es un preprocesador de CSS permite la utilización de variables, reutilización, herencia entre otras funciones que hacen más facil y legible la creación de hojas de estilo en cascada.

Sublime Text mi editor de texto favorito, todos mis proyectos los he creado con él. Permite agregar plugins para acelerar la escritura de código.

Git es un sistema de control de versiones el cual lo utilizó en todos mis proyectos y me permite tener un seguimiento total de la evolución de cada proyecto.
