Frameworks

Grandes y complejas aplicaciones requieren principalmente de código organizado y reutilizable es por ello que he aprendido por mis propios medios:

Symfony2: Es un Framework de PHP para el desarrollo de aplicaciones web que llamó mi atención, el gran interes por aprenderlo fue principalmente por la cantidad de patrones de diseño que implementa que hacen de las buenas practicas.

AngularJS es un Framework de Javascript, su objetivo es el desarrollo de aplicaciones sigle-page. La necesidad por aprenderlo fue porque me permitió crear formularios e interfaces de usuarios agradables y faciles de usar. Al implementa el patron MVC resulta mucho más facil mantener el codigo, realizar Tests y agregar nuevas funciones a la aplicación web.

Bootstrap es un Framework Front-end que contiene una colección de herramientas para el diseño de una aplicaciones web, la ventaja al utilizarlo es que me provee una estructura que se adapta a las diferentes resoluciones, "Responsive Design", además de traer consigo tipografias, iconos y componentes como Modals, Tabs, Navs, etc.

Ionic Framework me permitió programar aplicaciones nativas para dispositivos mobiles utilizando tecnologias web (HTML5, Javascript y CSS3), incluso poder hacer uso de todos los sensores del smartphone.
